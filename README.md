## 🚀 Quick Start

### Prerequisites

```bash
node >= 18.0.0
npm >= 9.0.0
1. Clone the repo
Bash

git clone https://github.com/ahmadrrrtx/brainrot-iq.git
cd brainrot-iq
2. Install dependencies
Bash

npm install
3. Set up environment variables
Bash

cp .env.example .env
Edit .env:

env

VITE_GROQ_API_KEY=your_groq_api_key
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
4. Start development
Bash

npm run dev
Open http://localhost:3000 🚀

5. Build for production
Bash

npm run build
🔑 Getting API Keys
Groq API (Free)
Go to console.groq.com
Sign up → Create API key
Copy key → paste in .env
Supabase (Free tier)
Go to supabase.com
Create new project
Run this SQL in SQL Editor:
SQL

create table scores (
  id uuid default gen_random_uuid() primary key,
  name text not null,
  score integer not null,
  level text,
  emoji text,
  created_at timestamptz default now()
);

-- Enable RLS
alter table scores enable row level security;

-- Allow public reads
create policy "Public read" on scores
  for select using (true);

-- Allow public inserts
create policy "Public insert" on scores
  for insert with check (true);

-- Index for performance
create index scores_score_idx on scores(score desc);
create index scores_created_at_idx on scores(created_at desc);
Copy URL + anon key → paste in .env
🌐 Deploy to Vercel
Option 1: GitHub Integration (Recommended)
Push your code to GitHub
Go to vercel.com → Import project
Select your repo
Add environment variables in Vercel dashboard:
VITE_GROQ_API_KEY
VITE_SUPABASE_URL
VITE_SUPABASE_ANON_KEY
Deploy → Auto-deploys on every commit ✅
Option 2: Vercel CLI
Bash

npm i -g vercel
vercel login
vercel --prod
